# H3rmel's ESLint config for NextJS

My personal ESLint configuration for NextJS projects.

## Setup

Install the package:

```bash
npm i --save-dev @h3rmel/eslint-config-next
```

Extends the configuration:

```js
// .eslintrc.js
module.exports = {
  extends: ["@h3rmel/eslint-config-next"],
};
```

And enjoy it!
