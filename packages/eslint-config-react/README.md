# H3rmel's ESLint config for React

My personal ESLint configuration for React projects.

## Setup

Install the package:

```bash
npm i --save-dev @h3rmel/eslint-config-react
```

Extends the configuration:

```js
// .eslintrc.js
module.exports = {
  extends: ["@h3rmel/eslint-config-react"],
};
```

And enjoy it!
